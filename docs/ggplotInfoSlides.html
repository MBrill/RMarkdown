bersicht ber ggplot

  • ggplot implementiert die grammar of graphics ( (Wilkinson 2005)).
  • Eine Visualisierung ist ein Abbildung (im mathematischen Sinn), eine Beziehung zwischen den Daten und den visuellen Eigenschaften in einer grafischen Ausgabe.
  • Daten sind qualitativ oder quantitativ,
  • die visuellen Eigenschaften der Daten werden durch die x- oder y-Koordinaten eines Punkts, die verwendete Farbe, die Hhe von Balken und vieles mehr beschrieben.

Ein erstes Beispiel

ggplot(liefer) + 
    geom_bar(mapping=aes(x=Lieferzeiten))

ggplot geht davon aus, dass die Daten im wide-Format vorliegen. Das bedeutet, dass jede Zeile ein Individuum beschreibt, mit den Merkmalen als Spalte. Dies ist in unserem kleinen Beispielen der Lieferzeiten der Fall, so dass wir ggplot direkt anwenden knnen.

Vernderungen

bars <- ggplot(liefer) + 
           geom_bar(mapping=aes(x=Lieferzeiten),
                    fill=myPalette[3])

bars

Formatierung der x-Achse

bars +
  scale_x_discrete(limits=c(1,2,3,4,5,6,7))

Beschriftungen

bars + 
  scale_x_discrete(limits=c(1,2,3,4,5,6,7)) +
  labs(
    title="Absolute Hufigkeiten der Lieferzeiten", 
    x="Tage", 
    y="Absolute Hufigkeiten"
  )

Relative Hufigkeiten

ggplot(liefer) +
  geom_bar(mapping=aes(x=Lieferzeiten, y = ..prop..), 
           fill=myPalette[3]) +
  scale_x_discrete(limits=c(1,2,3,4,5,6,7)) +  
  labs(
    title="Relative Hufigkeiten der Lieferzeiten", 
    x="Tage", 
    y="Anteile"
  )

Vertikale und horizontale Ausgaben

  • Vertikale und horizontale Ausgaben erzeugen wir mit coord_flip().

Box-Plots

ggplot(liefer)  +
  geom_boxplot(mapping=aes(x="", y = Lieferzeiten, fill=myPalette[2])) +
  labs(
    title="Box-Plot der Lieferzeiten", 
    x="", 
    y="Absolute Hufigkeiten"
  ) +
  coord_flip() +
  theme(legend.position = "none")

Verteilungsfunktionen

ggplot(liefer, aes(x=Lieferzeiten)) + 
  stat_ecdf(geom = "step", pad=TRUE, color=myPalette[3], size=2) +
  scale_x_discrete(limits=c(1,2,3,4,5,6,7)) +
  labs(
    title="Empirische Verteilungsfunktion der Lieferzeiten", 
    x="Tage", 
    y="Kumulierte relative Hufigkeiten"
  )

Der Datensatz Krpergre

ggplot(frauen)  +
  geom_bar(mapping=aes(x=Koerpergroesse),
           fill=myPalette[3]) +
  scale_x_continuous(limits=c(150.0, 185.0)) +
  labs(
    title="Absolute Hufigkeiten der Krpergroe von Frauen", 
    x="Krpergre in cm", 
    y="Hufigkeiten"
  )

Histogramm

ggplot(frauen) + 
  geom_histogram(mapping=aes(x=Koerpergroesse), closed="left", 
          fill=myPalette[3], color="black") +
  scale_x_continuous(limits=c(150.0, 185.0)) + 
  labs(
    title="Krpergre von Frauen", 
    subtitle="Default-Anzahl von Klassen",
    x="Krpergre in cm",
    y="Hufigkeiten der Klassen"
  ) 
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Bessere Wahl der Klassenbreite

ggplot(frauen) + 
  geom_histogram(mapping=aes(x=Koerpergroesse), closed="left", 
          breaks=seq(152.0, 182.0, 3.0),
          fill=myPalette[3], color="black") +
  scale_x_continuous(limits=c(150.0, 185.0)) + 
  labs(
    title="Krpergre von Frauen", 
    subtitle="Verwendete Klassenbreite: 3 cm zwischen 150 cm und 185 cm",
    x="",
    y="absolute Hufigkeit"
  )
